I am a software technologist working on the [TileDB](https://www.tiledb.com) database. In the past, I was a research engineer building tools for surgical navigation using [MRI](https://golbylab.bwh.harvard.edu/), [mass spectrometry](https://agarlab.bwh.harvard.edu/) and other technologies. For several years, I was also heavily involved with [the Julia Language](https://julialang.org). Long ago, I ran instrument systems and supported researchers on an [ice breaker](https://en.wikipedia.org/wiki/Nathaniel_B._Palmer_(icebreaker)) with the US Antarctic Program.

## Open Source

- Julia Language: core committer (2013-2016). I built [Clang.jl](https://github.com/JuliaInterop/Clang.jl), which is still actively maintained and used to generate the low-level C bindings for many packages. Also [contributed](https://github.com/JuliaLang/julia/commits?author=ihnorton) to the core project, and across the ecosystem, answered many questions on the [mailing list](https://groups.google.com/g/julia-users/search?q=from%3Aisaiah), and learned from some [amazing people](https://github.com/JeffBezanson/phdthesis/blob/876be73a5aab9b034fac3eb9ea9d8f96713f786f/acks.tex).
- 3D Slicer: contributed to [3D Slicer application](https://github.com/Slicer/Slicer/commits?author=ihnorton), [SlicerDMRI](https://github.com/SlicerDMRI/SlicerDMRI/commits?author=ihnorton) extension for diffusion MRI and tractography, [OpenIGTLink](https://github.com/openigtlink/OpenIGTLink/commits?author=ihnorton), and the [PLUS Toolkit](https://github.com/PlusToolkit/PlusLib/commits?author=ihnorton). [Built](https://github.com/SlicerDMRI/SlicerDMRI/commit/d610ab8b03f4a47d37e34da8d2b63c2b38b10c45) support for the DICOM tractography standard in SlicerDMRI.
- Various contributions to: [CMake](https://github.com/Kitware/CMake/commits?author=ihnorton), [Notepad++](https://github.com/notepad-plus-plus/notepad-plus-plus/commit/556901b83b1d71e47c861413ab7577693fdf5c9a), [Pixi](https://github.com/prefix-dev/pixi/commits?author=ihnorton), [OpenSlide.jl](https://github.com/ihnorton/OpenSlide.jl), [vcpkg](https://github.com/MicrosoftDocs/vcpkg-docs/pull/139), [dask](https://github.com/dask/dask/pull/4679), [fwf_sequence_tools](https://github.com/filip-szczepankiewicz/fwf_sequence_tools/commits?author=ihnorton), and others.

## Sketches

- https://github.com/ihnorton/lldb.sh -- tiny LLDB plugin to pipe command output through a shell command (such as grep) within the LLDB repl.
- https://github.com/ihnorton/mojo-ffi -- FFI demos for the [Mojo language](https://www.modular.com/mojo), including some minor linker hackery to create shared libraries from Mojo, and link Mojo code against a C library (both unsupported within Mojo itself, when this was written).
- https://github.com/ihnorton/vtk-wrap-demo.jl -- Proof-of-concept project to call C++ from pure Julia, using Clang.jl to parse C++ headers and generate pure-Julia vtables (plus some macro fun).
- https://github.com/ihnorton/HackThatBase.jl -- Tool to patch new versions of Julia core library code over the prebuilt system image (long-since obsoleted, thankfully!)
- https://github.com/ihnorton/COMCall.jl -- simple COM calls in Julia, by hand.
- https://github.com/ihnorton/CSharp-Cpp-Array-Example -- Exploring C# to C(++) FFI
